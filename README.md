
# This project was built with Python Django and React js

This is a student management system where you can perform CRUD operations, ie create, read, update and delete informations in just a few clicks, making your work life easier. To do that you need to, Login or Create account. You can also view class information and the number of students in each class. This web app is a fully working sample that can be edited and scaled with more features to fit any management needs in any business field. Web based systems help businesses work from anywhere, even with just your smartphone and internet.
